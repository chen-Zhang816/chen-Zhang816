## 我的工作方式

**问题优先**：从真实用户痛点和证据出发，再决定做什么、不做什么。

**以做促想**：用 coding agent 做原型，在投入团队时间之前先验证想法。

**数据+叙事**：把量化信号和背后的人的故事结合起来。

**把质量做成体系**：评测集、标注标准、RAG / SFT / RL，而不是凭感觉。
